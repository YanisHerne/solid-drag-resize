# solid-drag-resize

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

Simple, flexible library for creating draggable and resizable components in SolidJS. Similar API to [react-rnd](https://github.com/bokuweb/react-rnd)

## Quick start

Install it:

```bash
npm i solid-drag-resize
# or
yarn add solid-drag-resize
# or
pnpm add solid-drag-resize
```

Use it:

```tsx
import { DragAndResize } from "solid-drag-resize"
```
